Cendra syncs properties from your property management system and provides a detailed view of each property's configuration, amenities, and channel status. Every property synced to Cendra becomes a source of knowledge for your AI agents.

## Property Detail Sections

Each property in Cendra has the following configurable sections:

| Section | What it contains |
|---------|-----------------|
| **Descriptions** | Property name, headline, detailed description, house rules |
| **Location** | Address, coordinates, access instructions, neighborhood details |
| **Amenities** | Full list of property facilities and amenities |
| **Photos** | Cover photo and photo gallery |
| **Rooms & Beds** | Room configuration, bed types, capacity |
| **Pricing** | Base nightly rate, seasonal rates, discounts |
| **Availability** | Default availability, booking window, advance notice |
| **Booking Rules** | Min/max stay, preparation time, check-in restrictions |
| **Cancellation Policies** | Cancellation rules and deadlines |
| **Channel Integrations** | Sync status across connected booking channels |
| **Taxes** | Tax configuration per property |
| **Fees** | Additional fees (cleaning, service, etc.) |
| **Markups** | Channel-specific pricing adjustments |
| **WiFi Access** | Network name and password for guest sharing |
| **Custom Fields** | Property-specific metadata |
| **Owner License** | Legal documentation and licensing info |

## How Properties Feed AI Agents

Everything in a property's profile becomes part of the AI agent's knowledge:
- Guest asks about WiFi → AI retrieves the WiFi password from the property profile
- Guest asks about amenities → AI references the property's amenity list
- Guest asks about check-in → AI uses the property's location and access instructions
- Guest asks about pricing → AI references the property's rate configuration

## Property Sync

Properties sync automatically from your PMS. When you add, update, or remove a property in your PMS, the change appears in Cendra within minutes. No manual import required.

## Channel Integration Status

For each property, Cendra shows the sync status across your connected booking channels (Airbnb, Booking.com, Vrbo, etc.). This helps you verify that property data is consistent across all platforms.
