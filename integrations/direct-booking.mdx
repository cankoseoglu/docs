Cendra includes a customizable direct booking site where guests can book your properties directly — bypassing OTA commissions from Airbnb, Booking.com, and other platforms. Drive direct bookings and keep more revenue.

## What You Can Customize

| Setting | Description |
|---------|-------------|
| **Custom domain** | Use your own domain (e.g., book.yourcompany.com) |
| **Logo & favicon** | Brand the booking site with your logo |
| **Hero content** | Customize the main banner and messaging |
| **SEO settings** | Page title, meta description for search visibility |
| **Social links** | Connect social media accounts |
| **FAQs** | Add frequently asked questions to the booking page |

## Booking Configuration

| Setting | Description |
|---------|-------------|
| **Deposit policies** | Configure deposit requirements for direct bookings |
| **Check-in rules** | Set check-in/out times and restrictions |
| **Payment methods** | Accept payments via Stripe |
| **Booking rules** | Minimum stay, advance notice, availability windows |

## Why Direct Bookings Matter

| Channel | Typical commission |
|---------|-------------------|
| Airbnb | 3-15% host fee |
| Booking.com | 15-20% commission |
| Vrbo | 5-8% host fee |
| **Direct (via Cendra)** | **0% commission** |

## How It Works With AI Agents

When a guest inquires via your direct booking site, the conversation flows into Cendra's unified inbox. Your AI agent handles the inquiry using the same pipeline as OTA bookings — with the added benefit of zero commission on the resulting reservation.

Cendra's AI can also suggest your direct booking site to returning guests, encouraging them to book directly next time.
