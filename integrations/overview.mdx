Cendra integrates with 45+ property management systems to sync properties, reservations, guest data, pricing, and availability in real time. Once connected, Cendra's AI agents use your PMS data to respond to guest messages accurately and automatically across WhatsApp Business, Gmail, and Outlook.

## How PMS Integration Works

1. **Connect** — Enter your PMS API credentials in Cendra's onboarding flow (under 5 minutes)
2. **Sync** — Cendra imports your properties, reservations, and guest data automatically
3. **Stay current** — Data syncs in real time — new bookings, cancellations, and changes appear immediately
4. **AI uses it** — Every AI response is grounded in live PMS data, not stale information

## What Syncs

| Data | Description |
|------|-------------|
| **Properties** | Name, address, amenities, descriptions, photos, room configuration |
| **Reservations** | Check-in/out dates, guest count, status, special requests, booking source |
| **Guest profiles** | Name, email, phone number, language preference |
| **Pricing** | Nightly rates, seasonal rates, fees, discounts, taxes |
| **Availability** | Calendar availability, booking rules, restrictions |
| **Booking rules** | Minimum/maximum stay, advance notice, preparation time |
