Cendra integrates with Mr Alfred to sync properties, reservations, guest details, and availability. Once connected, Cendra's AI agents automatically respond to guest messages using live booking data from Mr Alfred — including check-in times, house rules, and property amenities.

## What Syncs From Mr Alfred

| Data | Details |
|------|---------|
| **Properties** | Name, address, amenities, descriptions, photos, room configuration |
| **Reservations** | Check-in/out dates, guest count, status, special requests, booking source |
| **Guest profiles** | Name, email, phone number, language preference |
| **Pricing** | Nightly rates, seasonal rates, fees, discounts, taxes |
| **Availability** | Calendar availability, min/max stay, advance notice |
| **Booking rules** | Preparation time, check-in restrictions, length of stay limits |

<Note>Exact fields depend on Mr Alfred API capabilities. Core reservation and property data syncs for all PMS integrations.</Note>

## How It Works

1. **Connect** — Select Mr Alfred during Cendra onboarding and authenticate your account
2. **Auto-sync** — Cendra imports your properties and active reservations
3. **Real-time updates** — Booking changes sync automatically
4. **AI responds** — Guest messages are answered using live Mr Alfred data

## What Cendra Can Do With Mr Alfred Data

| Scenario | How Cendra uses Mr Alfred data |
|----------|-------------------------------|
| Guest asks about check-in | AI retrieves check-in time and instructions from Mr Alfred property details |
| Guest requests early check-in | AI checks availability and offers upsell or escalates per your rules |
| Guest asks about amenities | AI references the property's amenity list synced from Mr Alfred |
| Pre-arrival message | AI sends check-in instructions using reservation dates and property address |
| Guest asks about pricing | AI references nightly rate and fees from Mr Alfred booking |

## Communication Channels

Once Mr Alfred is connected, Cendra handles guest messaging across:
- **WhatsApp Business** — automated two-way messaging
- **Gmail** — AI reads and responds to guest emails
- **Outlook** — AI reads and responds to guest emails
- **OTA & PMS Inbox** — messages from Airbnb, Booking.com, Vrbo, and other platforms via your PMS

## Setup Time

Under 5 minutes. Select Mr Alfred from the PMS dropdown during onboarding and authenticate your account.
