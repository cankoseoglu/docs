Cendra integrates with Slack to send real-time notifications about guest conversations, escalations, and important events directly to your team's Slack channels. Stay informed without keeping the Cendra inbox open.

## What Cendra Sends to Slack

| Notification type | When it's sent |
|------------------|---------------|
| **Escalations** | When a conversation is escalated from AI to human |
| **New messages** | When important guest messages arrive |
| **Operational alerts** | Device offline, sync issues, system notifications |

## How It Works

1. Connect your Slack workspace in Cendra settings
2. Select which Slack channel receives notifications
3. Configure which events trigger Slack messages
4. Receive real-time alerts as they happen

## Use Case

Slack integration is ideal for teams that:
- Monitor guest communication from Slack rather than switching apps
- Need instant escalation alerts for on-call property managers
- Want operational notifications (device offline, sync failures) in a shared channel
