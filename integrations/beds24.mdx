Cendra integrates with Beds24 to sync properties, reservations, guest details, and availability. Once connected, Cendra's AI agents automatically respond to guest messages using live booking data from Beds24 — including check-in times, house rules, and property amenities.

## What Syncs From Beds24

| Data | Details |
|------|---------|
| **Properties** | Name, address, amenities, descriptions, photos, room configuration |
| **Reservations** | Check-in/out dates, guest count, status, special requests, booking source |
| **Guest profiles** | Name, email, phone number, language preference |
| **Pricing** | Nightly rates, seasonal rates, fees, discounts, taxes |
| **Availability** | Calendar availability, min/max stay, advance notice |
| **Booking rules** | Preparation time, check-in restrictions, length of stay limits |

<Note>Exact fields depend on Beds24 API capabilities. Core reservation and property data syncs for all PMS integrations.</Note>

## How It Works

1. **Connect** — Select Beds24 during Cendra onboarding and authenticate your account
2. **Auto-sync** — Cendra imports your properties and active reservations
3. **Real-time updates** — Booking changes sync automatically
4. **AI responds** — Guest messages are answered using live Beds24 data

## What Cendra Can Do With Beds24 Data

| Scenario | How Cendra uses Beds24 data |
|----------|-------------------------------|
| Guest asks about check-in | AI retrieves check-in time and instructions from Beds24 property details |
| Guest requests early check-in | AI checks availability and offers upsell or escalates per your rules |
| Guest asks about amenities | AI references the property's amenity list synced from Beds24 |
| Pre-arrival message | AI sends check-in instructions using reservation dates and property address |
| Guest asks about pricing | AI references nightly rate and fees from Beds24 booking |

## Communication Channels

Once Beds24 is connected, Cendra handles guest messaging across:
- **WhatsApp Business** — automated two-way messaging
- **Gmail** — AI reads and responds to guest emails
- **Outlook** — AI reads and responds to guest emails
- **OTA & PMS Inbox** — messages from Airbnb, Booking.com, Vrbo, and other platforms via your PMS

## Setup Time

Under 5 minutes. Select Beds24 from the PMS dropdown during onboarding and authenticate your account.
