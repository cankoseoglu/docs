Cendra integrates natively with Guesty to sync properties, reservations, guest details, and availability in real time. Once connected, Cendra's AI agents automatically respond to guest messages using live booking data from Guesty — including check-in times, house rules, and property amenities.

## What Syncs From Guesty

| Data | Details |
|------|---------|
| **Properties** | Name, address, amenities, descriptions, photos, room configuration |
| **Reservations** | Check-in/out dates, guest count, status, special requests, booking source (Airbnb, Booking.com, direct) |
| **Guest profiles** | Name, email, phone number, language preference |
| **Pricing** | Nightly rates, seasonal rates, fees, discounts, taxes |
| **Availability** | Calendar availability, min/max stay, advance notice |
| **Booking rules** | Preparation time, check-in restrictions, length of stay limits |

## How It Works

1. **Connect** — Enter your Guesty API key in Cendra's onboarding flow
2. **Auto-sync** — Cendra imports all properties, active reservations, and guest data
3. **Real-time updates** — New bookings, modifications, and cancellations sync automatically
4. **AI responds** — When a guest messages, the AI uses their Guesty reservation data to respond accurately

## What Cendra Can Do With Guesty Data

| Scenario | How Cendra uses Guesty data |
|----------|-----------------------------|
| Guest asks about check-in | AI retrieves check-in time and instructions from Guesty property details |
| Guest requests early check-in | AI checks availability and offers upsell or escalates per your rules |
| Guest asks about amenities | AI references the property's amenity list synced from Guesty |
| Pre-arrival message | AI sends check-in instructions using reservation dates and property address |
| Guest asks about pricing | AI references nightly rate and fees from Guesty booking |

## Communication Channels

Once Guesty is connected, Cendra handles guest messaging across:
- **WhatsApp Business** — automated two-way messaging
- **Gmail** — AI reads and responds to guest emails
- **Outlook** — AI reads and responds to guest emails
- **OTA & PMS Inbox** — messages from Airbnb, Booking.com, Vrbo, and other platforms via your PMS
