Cendra connects to 45+ property management systems to sync properties, reservations, guest data, pricing, and availability in real time. Connecting your PMS is the foundation of everything Cendra does — AI agents use this data to respond to guests accurately.

<Info>**Quick link:** Connect your PMS in Cendra — https://app.cendra.ai/integrations/pms</Info>

## How to Connect

1. **Open onboarding** — after creating your workspace, Cendra guides you to the PMS connection step
2. **Select your PMS** — choose from Guesty, Hostaway, or Lodgify (native), or select from 40+ additional PMS in the dropdown
3. **Enter credentials** — provide your API key or authenticate via OAuth depending on your PMS
4. **Wait for sync** — Cendra imports your properties, reservations, and guest data. This typically takes 1-3 minutes depending on the number of properties

## Native Integrations

These PMS have direct API integrations with Cendra for the fastest sync:

| PMS | Authentication | Sync Speed |
|-----|---------------|------------|
| [Guesty](/integrations/guesty) | API key | Real-time |
| [Hostaway](/integrations/hostaway) | API credentials | Real-time |
| [Lodgify](/integrations/lodgify) | API key | Real-time |

## 40+ Additional PMS

Select your PMS from the dropdown during onboarding. Cendra supports Cloudbeds, OwnerRez, Beds24, Hospitable, Uplisting, Smoobu, Mews, and [40+ more](/integrations/overview).

## What Gets Synced

| Data | Description |
|------|-------------|
| **Properties** | Name, address, amenities, descriptions, photos, rooms |
| **Reservations** | Check-in/out dates, guest count, status, booking source |
| **Guest profiles** | Name, email, phone, language preference |
| **Pricing** | Nightly rates, seasonal rates, fees, taxes |
| **Availability** | Calendar, booking rules, restrictions |

## After Connecting

Once your PMS is connected, Cendra automatically:
- Builds a knowledge base from your property data
- Syncs new bookings, modifications, and cancellations in real time
- Makes reservation data available to AI agents for guest responses
