Cendra's Dashboard provides real-time analytics on AI automation performance, guest communication metrics, active issues, and escalation rates. Monitor how your AI agents are performing across all properties from a single view.

## Dashboard Components

### Active Issues
A summary card showing the number of open issues that need attention — messages the AI escalated, missing knowledge requests, and unresolved guest problems.

### Automation Trends
A chart showing AI automation performance over time:
- **Automation rate** — percentage of messages handled by AI without human intervention
- **Semi-automated** — messages where AI drafted a response for human approval
- **Manual** — messages that required full human response

### Escalation Chart
Track how often and why conversations are escalated to your team:
- Escalation frequency over time
- Most common escalation triggers
- Resolution time for escalated conversations

### Property Automation Grid
Per-property breakdown of AI performance:
- Automation rate by property
- Message volume by property
- Escalation rate by property

### Date Range Filtering
Filter all dashboard metrics by custom date ranges to analyze specific periods.

## Key Metrics

| Metric | What it measures |
|--------|-----------------|
| **Automation rate** | % of messages handled by AI without human help |
| **Escalation rate** | % of messages escalated to human team |
| **Response time** | Average time between guest message and response |
| **Active issues** | Number of unresolved items needing attention |
| **Messages handled** | Total messages processed by AI |
| **Knowledge gaps** | Questions the AI couldn't answer (missing knowledge) |

## Using the Dashboard

- **Monitor daily** — check active issues and automation rate each morning
- **Identify patterns** — if escalation rate spikes, investigate which properties or topics are causing it
- **Improve knowledge** — dashboard highlights knowledge gaps. Add missing information to improve automation rate.
- **Track progress** — watch automation rate increase over time as you refine your AI agents
