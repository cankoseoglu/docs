Cendra automatically creates unique smart lock access codes for each guest reservation and revokes them at checkout. No manual code management — Cendra handles the entire lifecycle based on your reservation data from your PMS.

## How Guest Access Codes Work

1. **New reservation syncs from PMS** — Cendra detects a confirmed booking with check-in/out dates
2. **Code created** — Cendra generates a unique access code on the property's smart lock via Seam
3. **Code activated** — the code becomes active at the check-in time
4. **Guest arrives** — they use the code to unlock the door (you can share the code via WhatsApp or email using AI agents)
5. **Code revoked** — at checkout time, Cendra automatically removes the code from the lock

## Code Lifecycle

| Stage | When | What happens |
|-------|------|-------------|
| **Created** | Reservation confirmed | Unique code generated and scheduled on the lock |
| **Active** | Check-in time | Code works on the lock |
| **In use** | During stay | Guest can lock/unlock with their code |
| **Revoked** | Check-out time | Code is automatically removed from the lock |

## Sharing Codes with Guests

Cendra's AI agents can share access codes with guests automatically through your communication channels:

- **WhatsApp** — AI sends the code as part of pre-arrival instructions
- **Email** — included in check-in information emails
- **Timed delivery** — codes can be shared 24 hours before check-in, or at any time you configure

The AI agent references the access code from the property's device data — it's part of the knowledge the AI uses when responding to check-in questions.

## Security

- **Unique codes per reservation** — every guest gets a different code
- **Time-bound** — codes only work during the reservation period
- **Automatic cleanup** — no forgotten active codes from past guests
- **Audit trail** — Cendra logs when codes are created, used, and revoked
- **Real-time status** — see which codes are currently active on each lock

## Supported Locks

Guest access codes work with any Seam-compatible smart lock that supports temporary codes, including:
- Yale smart locks and deadbolts
- August smart locks
- Schlage smart locks
- Level Lock
- Igloohome locks and lockboxes
- Kwikset smart locks
- Nuki smart locks
- And 15+ additional brands

## Combining with AI Agents

The most powerful setup combines guest access codes with Cendra's AI agents:

1. **Guest asks "how do I get in?"** → AI retrieves the access code and property entry instructions from the knowledge base
2. **Pre-arrival message** → AI automatically sends check-in instructions with the access code 24 hours before arrival
3. **Guest reports lock issue** → AI troubleshoots (battery check, code verification) and escalates to your team if needed
