Cendra integrates with smart locks and IoT devices via the Seam platform to automate guest access, monitor device status, and manage property hardware alongside your guest communication. This makes Cendra one of the few AI guest communication platforms that also handles physical property access.

<Info>**Quick link:** Open Home Automation in Cendra — https://app.cendra.ai/home-automation</Info>

## What Cendra Can Do With Smart Locks

- **Automatic guest access codes** — generate unique access codes for each reservation, tied to check-in and check-out dates
- **Real-time device monitoring** — see lock status (locked/unlocked), battery level, and online/offline status
- **Automatic code revocation** — access codes are revoked when the guest checks out
- **Instant notifications** — get alerts when devices go offline, batteries are low, or locks are tampered with

## How It Works

1. **Connect your Seam account** in Cendra settings
2. **Cendra discovers your devices** — all connected locks and devices sync automatically
3. **Assign devices to properties** — link each device to the correct property
4. **Automation kicks in** — Cendra creates guest access codes, monitors devices, and sends alerts

## Real-Time Monitoring

Cendra receives real-time updates from your devices:

| Event | What happens |
|-------|-------------|
| Device connected | New device appears in Cendra |
| Device disconnected | Alert sent — device went offline |
| Lock locked/unlocked | Status updated in real-time |
| Access code set | Confirmation that guest code is active |
| Access code removed | Confirmation that guest code is revoked |
| Battery status changed | Battery level updated, low-battery alerts |
